Formspree로 네이버메일(dev@eden-works.co.kr) 수신 설정하기
1) https://formspree.io 가입 → 이메일(dev@eden-works.co.kr) 확인
2) New Form 생성 → Endpoint URL(예: https://formspree.io/f/abcdwxyz) 복사
3) index.html 의 action 값을 해당 Endpoint 로 교체
4) 저장소 업로드 후 테스트 제출 → 받은편지함에서 수신 확인
참고: 무료 플랜 전송량 제한이 있으니 필요에 따라 유료/대체 서비스(Web3Forms/Getform/Basin 등) 고려
